## 二、 “简阅”项目各组件映射落地解析

1. **服务器（鲲鹏TaiShan 200 - ARM64）**：
   - 项目后端基于多架构兼容的 `python:3.12-slim` Docker 镜像构建，在编译阶段和运行阶段完全适配华为鲲鹏 920 处理器 ARM64 指令集。
2. **虚拟机与系统（openEuler 22.03 LTS SP2）**：
   - 虚拟机提供 2vCPU / 4GB 内存 / 40GB系统盘。作为容器宿主，搭载安全可信、性能卓越的国产操作系统 openEuler 22.03，保障底层运行安全。
3. **容器引擎（Docker CE 24.x）**：
   - 配置有自启动及日志轮转规则（最大10MB，保留3个），防范控制台日志无限期膨胀导致虚拟机 40GB 磁盘满载故障。
4. **编排工具（Docker Compose v2.20+）**：
   - 项目通过根目录 `docker-compose.yml` 统一生命周期管理，并通过 `ops/deploy_yue.sh` 脚本自动拉起，实现一键极速重构。
5. **数据库（MySQL 8.0.x 与 SQLite）**：
   - 项目本地开发和轻量测试默认运行嵌入式 SQLite 3.x（保存在挂载数据卷 `yue_data` 中的 `yue.db`），符合零依赖启动要求。
   - 由于系统数据访问层基于 `SQLAlchemy ORM` 编写，生产信创合规化部署时，可随时通过修改环境变量 `DATABASE_URL` 无缝切换至标准的 **MySQL 8.0.x**（或国产达梦、GaussDB 等数据库）作为生产级多节点持久化存储。
6. **反向代理（Nginx 1.24.x）**：
   - Nginx 反代配置于系统最前端，负责外部 SSL 证书安全卸载（实现 443 HTTPS 加密），并将路由分发请求向后反向代理至容器的 `8000` 端口。
